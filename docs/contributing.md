# Как внести вклад в Tomodoro

Спасибо за интерес к проекту! Ниже — краткие правила и рекомендации.

## Быстрый старт

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feat/короткое-описание`
3. Сделайте изменения и добавьте тесты
4. Запустите тесты локально: `php artisan test` / `vendor/bin/phpunit`
5. Откройте Pull Request и дождитесь ревью

## Стандарты кода

- PHP: PSR-12
- Коммиты: рекомендуем Conventional Commits (например: `feat: add export to CSV`)
- Покрытие тестами: добавляйте тесты для большинства изменений бизнес-логики

## Линтеры и проверки

- PHPStan/psalm (если настроены)
- phpcs для стиля

## PR Checklist

- [ ] Тесты добавлены / актуализированы
- [ ] Документация обновлена при необходимости
- [ ] Все CI проверки проходят
