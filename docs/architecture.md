# Архитектура проекта

Tomodoro построен на Laravel + Livewire. Ниже — основные компоненты системы.

## Компоненты

- Backend: Laravel (MVC) — обработка данных, API, миграции, задачи по расписанию.
- Frontend: Blade + Livewire — интерактивные компоненты без полного SPA.
- Очереди: Laravel Queue (рекомендуется Redis).
- Хранилище: MySQL/Postgres для данных, `storage/` для файлов.

## Важные модели

- `User` — пользователь приложения
- `Task` — задача
- `PomodoroSession` — запись Pomodoro-сессии
- `CalendarEvent` — события календаря
- `Tag` — теги для задач

## Сервисы

В папке `app/Services/` находятся бизнес-сервисы (например, экспорт/импорт данных, логика Pomodoro).

## Livewire компоненты

- `CalendarOverview` — обзор календаря
- `PomodoroTimer` — компонент таймера
- `PresetManager` — управление пресетами таймера
- `TodoManager` — управление задачами

## Поток данных (пример: создание Pomodoro-сессии)

1. Пользователь запускает таймер в `PomodoroTimer`.
2. Livewire компонент отправляет AJAX-запрос на бэкенд.
3. Бэкенд сохраняет `PomodoroSession`, возможно добавляет события и статистику.
4. События (events) и слушатели могут обновлять агрегации/статистику асинхронно через очереди.
