# Руководство пользователя

Добро пожаловать в Tomodoro — краткий справочник по использованию приложения.

## Основные понятия

- Pomodoro-сессия — интервал работы (обычно 25 минут) и короткий перерыв.
- Пресеты — готовые настройки таймера (время работы и перерывов).
- Задача — единица работы с названием, описанием, категорией и тегами.

## Запуск таймера

1. Выберите задачу или создайте новую.
2. Выберите пресет или настройте длину сессии вручную.
3. Нажмите «Старт» — таймер начнёт обратный отсчёт.
4. По окончании сессии можно отметить задачу как выполненную или продолжить.

### Горячие клавиши (по умолчанию)
- Пробел — старт/пауза таймера
- R — сброс таймера
- N — новая задача

## Управление задачами

- Создание: название, категория, теги, примерная длительность.
- Редактирование/удаление: через интерфейс списка задач.
- Фильтрация: по тегам, категориям и статусу.

## Отчёты и статистика

- Просмотр истории Pomodoro-сессий по периодам (день, неделя, месяц).
- Экспорт истории в CSV для анализа.

## Экспорт и импорт

- Экспорт: выгрузить текущие задачи и сессии в JSON/CSV.
- Импорт: загрузить ранее экспортированные данные.

## Настройки пользователя

- Темы оформления (светлая/тёмная).
- Уведомления (звуки и десктоп-уведомления).

## Частые вопросы

- "Как восстановить удалённую задачу?" — По умолчанию удалённые задачи без восстановления, используйте резервные копии.
